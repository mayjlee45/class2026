🧰 데이터 AI 도구 투어

ChatGPT·Claude·Gemini 말고도, 데이터를 다루는 무료 AI 도구가 많습니다. 버튼으로 열어 가벼운 미션 하나씩 해보세요.

🔒 어떤 도구든 공개·가공 데이터만. 실제 부대·장비·인명·군사정보는 절대 업로드·입력 금지(입력=외부 전송·학습에 쓰일 수 있음). 미션은 공개 데이터로만.
① 문서 이해 · Q&A

NotebookLM

notebooklm.google.com · 구글 계정 · 무료

PDF·문서 여러 개를 올리면 내용 기반으로 답변하고, 버튼 하나로 오디오 요약(AI 두 명이 팟캐스트처럼)까지 만들어 줍니다.

📌 사용법: 새 노트북 → Sources에 공개 PDF·기사·문서 추가 → 질문 입력. 오른쪽 Audio Overview → Generate로 오디오 요약 생성.
🎯 미션: 공개 문서(예: 데이터 관련 기사·규정 PDF) 1개 올려 → "핵심 3줄 요약" 질문 → 오디오 요약 1분 들어보기.

② 데이터 대화형 분석

Julius AI (무료 월 15회)

julius.ai · 무료 계정 · CSV/엑셀 업로드

표 데이터를 올리고 말로 분석을 시키면 통계와 차트를 자동으로 그려 줍니다. 어제 Orange로 클릭해 한 걸, 이번엔 대화로.

📌 사용법: 파일 업로드 → "가장 많은 항목을 막대그래프로", "결측값 몇 개인지 알려줘"처럼 한국어로 요청. (무료는 월 15회라 질문을 아껴서)
🎯 미션: 공개 CSV 하나 올려 → "이 데이터 요약해줘 + 눈에 띄는 관계 하나 그래프로".

③ 발표용 시각화

Datawrapper

datawrapper.de · 무료(넉넉) · 로그인

표를 붙여넣으면 깔끔한 차트·지도를 만들어 줍니다. 분석용이 아니라 '전달용' 그래프 — 보고서·발표에 바로 쓰는 품질.

📌 사용법: New chart → 데이터 붙여넣기 → 차트 종류 고르기 → 제목·색 다듬기 → 이미지/임베드로 내보내기.
🎯 미션: 간단한 표(예: 장비 5종 고장 건수)를 붙여넣어 막대차트 하나 만들고 이미지로 저장.

④ 스토리 시각화

Flourish

flourish.studio · 무료(공개 게시) · 로그인

움직이는 인터랙티브 차트·스토리(바 차트 레이스 등)를 코드 없이. 발표에서 시선을 끌 때.

📌 사용법: New visualization → 템플릿 선택 → Data 탭에 데이터 입력 → Preview로 확인 → Publish/Export. 무료는 게시물이 공개되니 공개 데이터만!
🎯 미션: 샘플 데이터로 막대 또는 바 차트 레이스 템플릿 하나 미리보기까지.
