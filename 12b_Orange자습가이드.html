방법 ① 설치 프로그램 (가장 쉬움 · 추천)

  1. orangedatamining.com/download 접속
  2. 본인 OS(Windows/Mac)용 설치 파일 내려받기
  3. 더블클릭 → 안내대로 다음·다음 → 끝. 바탕화면 아이콘 실행.

방법 ② 파이썬(아나콘다)에 익숙하다면

  • conda create -n eduenv python=3.10 -y
  • conda activate eduenv
  • conda install -c conda-forge orange3 -y → python -m Orange.canvas
막혀도 괜찮습니다 — 방법 ①이면 5분이면 됩니다. 설치만 되면 아래 워크플로를 그대로 따라올 수 있어요.
🧹 11차시 — 전처리 워크플로
1 데이터 열고 다듬기 (순서대로) ▶

위젯을 캔버스에 끌어다 놓고 점→점을 드래그해 선으로 연결합니다.

  1. File — 정비일지_학생용.xlsx 불러오기
  2. Data Table 연결 — 표로 눈으로 확인(문제가 몇 종류 보이나?)
  3. Feature Statistics — 컬럼별 결측·최소·최대 한눈에. 결측 총 102개 확인.
  4. Preprocess → Impute(대체) — 결측 102 → 0으로.
  5. Select Rows — 조건 소요시간 < 0 또는 = 9999 를 만들어 이상값 제외.
  6. (옵션) Apply Domain으로 무선기→무전기 통일, Continuize로 Y/N→1/0.
  7. Feature Statistics 다시 — 전·후 비교: 평균 소요시간 약 1006분 → 64분으로 정상화!
원본 파일은 절대 건드리지 않습니다 — Orange는 사본(워크플로)에서만 작업하니 안전합니다.
📊 12차시 — 시각화·분석 워크플로
2 그림으로 보고 인사이트 뽑기 ▶

전처리된 데이터에 시각화 위젯을 붙입니다.

  1. Distributions — 소요시간 분포(연속형 히스토그램) / 변수만 장비명으로 바꾸면 빈도그래프 → 무전기가 최다.
  2. Box Plot — 소요시간의 이상값(튀는 점) 확인.
  3. Scatter Plot — 소요시간 vs 정비비용(거의 직선!) / 장비연식 vs 소요시간(노후일수록 오래).
  4. Feature Statistics — 평균·중앙값·표준편차로 데이터 모양 읽기.
  5. Pivot Table — 행=장비명, 셀=소요시간 평균 → 장비별 비교. 행을 월로 바꾸면 계절 추이.
  6. Correlations — 수치 변수 상관. 연식↔소요시간 높게 나오지만… ⚠️ 상관 ≠ 인과(노후라는 숨은 원인)!
🎯 보너스 미션 — 혼자 해보기
3 8개 미션 풀이 (난이도 ★) ▶

★ 기본

  • 미래 날짜(2027) 찾기 — Select Rows: 정비일자 > 2026-12-31
  • 완료인데 0분 — Select Rows: 정비결과=완료 그리고 소요시간=0
  • 특정 부대만 — Select Rows: 부대=통신중대 → 건수 보기

★★ 중급

  • 무선기↔무전기 통일 — Apply Domain으로 값 이름 합치기
  • 전·후 행수·결측 비교 — Data Info 두 개를 전/후에 연결
  • 장비별 평균 소요시간 — Pivot Table(행=장비명, 평균)

★★★ 도전

  • 수치 변수 상관 보기 — Correlations: 높아도 인과는 아님(한 문장 메모)
  • 나만의 조건 — Select Rows로 궁금한 조건을 자유롭게 만들어 보기

4 막힐 때 팁 ▶
  • 위젯 연결이 안 되면 — 한 위젯의 오른쪽 점에서 다른 위젯으로 드래그하세요.
