🧪 AI 모델 직접 체험하기

네 가지 학습 방식을, 브라우저에서 직접 만져 보며 이해합니다. 각 버튼을 누르면 새 창에서 사이트가 열립니다.

💡 인터넷이 되는 PC(강사용 PC)에서 열립니다. 코딩·설치 없이 클릭만으로 동작해요. 강사가 시연하거나, 시간이 되면 직접 만져 보셔도 좋습니다.
① 지도학습 · 정답 보고 배우기

Teachable Machine

teachablemachine.withgoogle.com

웹캠으로 직접 가르치는 AI. 내가 예시(정답)를 보여주면 그걸로 배웁니다.

📌 이렇게: Get Started → Image Project → Standard. 클래스1에 '손바닥'을 웹캠으로 여러 장, 클래스2에 '주먹'을 여러 장 찍고 → Train → 카메라에 손을 바꿔 보이면 실시간으로 맞힙니다. "정답을 보고 배운다"가 눈앞에서 일어나요.

② 비지도학습 · 스스로 묶기

Visualizing K-Means

naftaliharris.com

정답 없이 비슷한 점끼리 군집으로 묶이는 과정을 한 단계씩 봅니다.

📌 이렇게: 위 메뉴에서 데이터 모양(예: Gaussian Mixture) 선택 → 중심점 배치(I'll Choose 또는 랜덤) → Update Centroids / Reassign Points를 번갈아 누르면 군집이 자리를 잡아갑니다.

③ 강화학습 · 보상으로 배우기

REINFORCEjs · Gridworld

cs.stanford.edu/people/karpathy

에이전트가 시행착오로 목표까지 가는 길을 스스로 학습하는 과정을 봅니다.

📌 이렇게: 페이지의 Toggle TD Learning(또는 Start)을 누르면 에이전트가 칸을 돌아다니며 각 칸의 가치를 점점 채웁니다. 색이 진해지는 칸 = 보상에 가까운 길. 잠시 두면 최적 경로를 찾아갑니다.

④ 트랜스포머 · 생성형 AI의 심장

Transformer Explainer

poloclub.github.io

브라우저에서 실제 GPT-2가 돌아가며, 다음 단어를 어떻게 예측하는지 보여줍니다.

📌 이렇게: 입력칸에 짧은 문장(예: Data is the new)을 넣으면 → 다음에 올 단어 후보와 확률이 뜹니다. 단어에 마우스를 올리면 어텐션(어느 단어에 집중하는지)이 선으로 보이고, Temperature 슬라이더로 답의 과감함도 조절됩니다.
