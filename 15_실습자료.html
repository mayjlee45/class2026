🔒 실데이터 금지. 여기 있는 가공 샘플만 사용하세요. 실제 부대·장비·인명·군사정보는 입력 = 외부 전송입니다(17차시).
실습 ① 비정형 → 정형 (회의록 → 표)

📝 회의 메모 (줄글 샘플)

오늘 정비반 주간회의 메모. 안테나 교체 건은 박상사가 맡기로 했고 6월 12일까지 하기로 함. 예비부품 신청은 누가 할지 아직 안 정해짐. 무전기 배터리 교체는 김중사가 이번 주 내로 처리. 광단말기 정기점검은 일정상 다음 주로 미룸. 야전교환기 펌웨어 업데이트는 지난주에 완료됨. 아, 회의록 정리 담당은 아직 안 정함.

⚡ 원샷 프롬프트

아래 회의 메모를 안건 / 담당 / 기한 / 상태 4열 표로 정리해줘. 메모에 없는 내용은 지어내지 말고 '미정' 또는 '-'로. 표만 출력해줘. [여기에 회의 메모 붙여넣기]
✅ 받은 뒤 — 원본 메모와 대조: 지어낸 것 없나? 빠뜨린 것 없나?
실습 ② 데이터 설명 → 메타데이터 초안

🗂️ 데이터 설명 (샘플)

이 데이터는 통신장비 정비일지야. 엑셀로 당직병이 매일 입력하고, 한 달에 약 3천 건 쌓여. 컬럼은 16개. 보안등급은 대외비고, 보존기간은 3년이야.

⚡ 원샷 프롬프트

아래 데이터 설명을 보고 메타데이터 표를 만들어줘. 열은 데이터명 / 생산주체 / 형식 / 갱신주기 / 보안등급 / 보존기간. 설명에 없는 항목은 '미정'으로. 표만 출력. [여기에 데이터 설명 붙여넣기]
6차시에서 손으로 쓴 양식 — 이제 AI와 함께. 결과는 실제와 맞는지 확인.
실습 ③ 긴 문서 → 내 상황 조항만 추출

📄 규정 발췌 (가상 샘플)

[통신장비 정비업무 규정(가상)] 제10조(정기점검) 모든 통신장비는 분기 1회 정기점검을 실시한다. 제11조(긴급정비) 작전에 영향을 주는 고장은 즉시 정비하고 당일 보고한다. 제12조(지연 보고) 정비가 예정일을 초과할 경우 24시간 이내에 사유와 함께 보고한다. 제13조(부품 교체) 부품 교체 시 교체 부품명·수량을 정비일지에 기재한다. 제14조(기록 보존) 정비일지는 3년간 보존한다. 제15조(외주 정비) 외주 정비 시 계약서와 결과보고서를 첨부한다. 제16조(폐기) 사용 불가 장비는 폐기심의를 거쳐 처리한다.

⚡ 원샷 프롬프트

아래 규정에서 '정비가 늦어져 지연 보고를 해야 하는 내 상황'에 적용되는 조항만 뽑아줘. 조항 번호와 원문을 그대로 인용하고, 추측하지 말고 이 문서 기준으로만. [여기에 규정 발췌 붙여넣기]
✅ AI가 없는 조항 번호를 지어내지 않았는지 — 원문과 대조하세요.
