📋 복붙 샘플 3종

아래 텍스트를 복사해서 ChatGPT / Claude / Gemini에 붙여넣으세요.

샘플 1. 회의록 → 표 정리
2026-06-15 정기 업무보고 회의 참석: 김팀장, 박주임, 이담당 1. 데이터 표준화 작업 - 박주임 담당, 7월 말까지 2. 메타데이터 작성 완료 - 이담당, 완료 3. 신규 양식 검토 - 김팀장, 미정 위 회의록을 안건/담당/기한/상태 4열 표로 정리해줘. 없는 건 지어내지 말고 '미정' 또는 '-'로.표만 출력.
샘플 2. 데이터 설명 → 메타데이터 초안
아래 데이터에 대한 메타데이터를 작성해줘. 형식: 표 (필드명 | 내용) 포함할 필드: 데이터명, 생산주체, 형식, 갱신주기, 보안등급, 항목정의 데이터 설명: - 이름: 월별 장비 가동률 현황 - 담당: 정비부대 운용팀 - 파일: Excel, 12개월치, 컬럼(장비번호/가동일수/총일수/가동률%) - 갱신: 매월 말 - 보안: 부대 내부 한정
샘플 3. 문서 요약 · 조항 추출
아래 지침의 핵심 내용을 다음 형식으로 정리해줘: 1) 목적 1문장 2) 담당자 의무사항 (번호 목록) 3) 위반 시 조치 (있다면) [지침 전문을 여기에 붙여넣기]
