2일차 · 공공데이터 분석 실습

📊 나의 공공데이터 분석 보고서

data.go.kr 같은 곳에서 데이터 하나를 받아 Orange로 분석하고, 그 결과를 아래 칸에 채워 PDF로 저장한 뒤 Padlet에 올리면 됩니다.

1. 받기data.go.kr에서 CSV·엑셀 파일 1개 내려받기
2. 분석Orange로 열어 전처리·통계·그래프
3. 기입아래 칸 채우고 그래프 캡처 첨부
4. 제출PDF로 저장 → Padlet 업로드
기본 정보
① 데이터 출처
사이트 이름과 데이터셋 제목, 가능하면 주소(URL)도.
② 데이터 개요 — Raw 확인
행 수 / 열 수 / 주요 변수(컬럼) 이름과 의미. (Orange의 Data Table·Feature Statistics에서 확인)
③ 던진 질문
분석의 목적 한두 줄. (예: 무엇과 무엇의 관계? 어디가 가장 큰가?)
④ 전처리 — 무엇을 어떻게
빈 값/이상한 값을 어떻게 했는지. (예: 결측은 평균으로 대치, 음수·0 등 불가능값은 제거)
⑤ 기술통계 결과
평균·중앙값·최댓값 등, 또는 그룹별 비교에서 눈에 띈 수치.
⑥ 시각화 — 그래프 캡처 첨부
산점도·히스토그램·박스플롯 등 스크린샷 1장. (캡처 후 아래에서 파일 선택)
↑ 이미지 파일을 선택하면 여기 미리보기가 뜹니다
⑦ 발견 · 인사이트
2~3가지. 관계가 있었나? 의외의 점은? (상관 ≠ 인과도 떠올려보세요)
⑧ 검증
📌 Padlet 제출
🔒 가공·공개 데이터만 사용하세요. 실제 부대·장비·인명·군사정보는 절대 입력·게시 금지(입력=외부 전송). · PDF 저장 방법: [PDF로 저장] → 프린터 목록에서 "PDF로 저장" 선택 → 저장. 그 파일을 Padlet에 올리면 됩니다.
