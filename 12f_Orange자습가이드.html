Orange3 자습 가이드

Orange3 노코드 도구로 전처리·시각화를 단계별로 따라해보세요.

🍊 Orange3 기본 워크플로우

아래 단계를 따라 Orange3로 정비일지를 분석해보세요.

Step 1: 데이터 열기
File 위젯 → Browse → 정비일지.csv 선택 → Data Table로 연결해 미리보기
Step 2: 품질 확인
Column Statistics 위젯 연결 → 결측 개수·이상값 확인
Step 3: 결측 처리
Impute 위젯 → 수치 칼럼: Mean 대체 / 범주 칼럼: Most Frequent 선택
Step 4: 시각화
Distributions → 소요시간 분포 확인 / Box Plot → 이상값 탐지 / Scatter Plot → 두 변수 관계
Step 5: 저장
Save Data 위젯으로 전처리된 데이터 저장
